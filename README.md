# ⚙️ DC Motor Control with Arduino

A simple but practical Arduino project that demonstrates how to control a DC motor using the L298N motor driver.  
The project allows you to control the **motor direction** and **speed** using PWM signals generated by the Arduino.

---

## 📌 Project Overview

This project shows how an Arduino can interface with an **L298N motor driver** to control a DC motor.  
Using digital and PWM pins, the system can:

- Change the **rotation direction**
- Control the **motor speed**
- Start and stop the motor smoothly

The goal of this project is to provide a clear and beginner-friendly example of **motor control using Arduino**.

---

## 🧰 Hardware Used

- Arduino board (Uno / Nano / similar)
- L298N Motor Driver
- DC Motor
- Power supply
- Jumper wires

---

## ⚡ Features

- PWM speed control  
- Direction control (forward / reverse)  
- Simple and easy-to-understand code
